# Chess

In this project, students implement a UCI chess engine. The engine is capable of playing chess games against human opponents. The engine is able to handle all standard chess rules, including en passant, castling, and pawn promotion. The student is encouraged to be creative with the engine's strategy.

Students also implement a chess GUI which is capable of human-human, human-engine, and engine-engine games. The GUI is able to display the chess board, highlight legal moves, and display captured pieces. The GUI is also able to display the game history and allow the user to navigate through it. The GUI works with any UCI-compatible chess engine.

The student engines compete against one another at a certain point in the program.

## Learning outcomes

- Algorithm design
- User interface design
- Creative confidence
