# Chess GUI

In this project, students implement a chess GUI which is capable of human-human, human-engine, and engine-engine games. The GUI is able to display the chess board, highlight legal moves, and display captured pieces. The GUI is also able to display the game history and allow the user to navigate through it. The GUI works with any UCI-compatible chess engine.

## Learning outcomes

- User interface design
- State management
