# Sudoku

This is a sudoku solver. The student is encouraged to be creative in designing the user interface and the algorithm for solving the puzzle. Ultimately, the student will be tested on the efficiency of their algorithm.

## Learning outcomes

- Algorithm design
- User interface design

## State of the project

Basic autoreview is working. The tests need a lot of refinement and the specification needs feedback. Some of my test code is available in [Python](https://gitlab.wethinkco.de/fen/pysudoku/) and [Java](https://gitlab.wethinkco.de/fen/sudoku-java).
