# Authenticate

In this project, students are asked to build a clone of an authentication service like Auth0. It must expose a JWT system for the client's app to use, and allow the end user to log in using federated logins like Google.

This is a complete product. It encompasses the entire software development lifecycle, from ideation to deployment, as well as both front-end and back-end work. The project is expected to be fully documented in high quality technical writing.

## Learning outcomes

- Authentication (JWT, OAuth2)
- Security
- Product development
- Cloud?
